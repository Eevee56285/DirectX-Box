# DirectX-Box
A simple recreation of the original Xbox operating system using Theseus OS, Xemu, and Linux.

DirectX-Box is a project that tries to rebuild the look and feel of the original Xbox dashboard while using modern open‑source tools. It is not just an emulator. It is a custom environment that mixes a recreated Xbox-style UI with Xemu for game compatibility.

## Core Parts
1. Theseus OS
   - Acts like the low-level system layer
   - Lets you build modular system components
   - Helps recreate Xbox-like system behavior

2. Xemu
   - Runs original Xbox games
   - Built into the system instead of being a separate app
   - Lets you launch games directly from the dashboard

3. Linux
   - The base system that everything runs on
   - Handles hardware, drivers, and GPU support
   - Makes the whole project portable and stable

## Features
- Custom Xbox-style dashboard
- Launch Xbox games through Xemu
- System apps like settings, file manager, etc.
- Optional “developer mode” for testing
- Themes and UI customization
- Boot into dashboard or Linux desktop

## Goal
DirectX-Box is meant to recreate the Xbox experience in a modern, open-source way while letting people learn about OS design, emulation, and system architecture.
