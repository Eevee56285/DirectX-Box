# DirectX-Box

DirectX-Box is a project where I’m rebuilding the classic Xbox experience using modern tools. The goal is to recreate the feeling of turning on an original Xbox, but without being tied to old hardware or the limitations of the real Xbox operating system. Instead of trying to perfectly duplicate the original software, this project focuses on capturing the style, the atmosphere, and the personality of the console while running on a modern Linux base.

The Theseus board is the heart of the system. It provides the dashboard, the animations, the layout, and the overall look that makes the whole thing feel like an Xbox. It recreates the green glow, the movement, and the structure of the original dashboard, but it also gives room for new ideas and improvements. When the system boots, Theseus is what you see first, and it becomes the main way you navigate everything.

Xemu is the part that actually runs the games. Instead of being treated like a separate emulator, it’s built directly into the system so it feels like a natural part of the experience. When you choose a game from the dashboard, it launches through Xemu, but the goal is for it to feel like the game is simply part of the system rather than something running in a separate window. It handles compatibility, virtual storage, and everything needed to make original Xbox games work smoothly.

Linux sits underneath everything and keeps the system stable. It handles the hardware, the drivers, the performance, and all the background tasks that make the whole setup reliable. It also gives the project flexibility, because it can run on different hardware without needing to be rebuilt from scratch. Linux also makes it possible to switch between the Xbox-style dashboard and a normal desktop environment whenever you want to work on the system or customize things.

DirectX-Box includes a full dashboard experience, game launching, system menus, settings, a file browser, theme support, and a simple developer mode for experimenting with the system. The goal is to make it feel like a real console, not just a Linux machine running an emulator. It’s meant to be something you can turn on and immediately feel that old-school Xbox energy, but with modern stability and features.

The reason this project exists is because the original Xbox had a very distinct style that nothing else really captured. The green aesthetic, the animations, the sound effects, and the whole “techno-organic” vibe gave it a personality that stood out. DirectX-Box is a way to bring that feeling back in a modern way. It’s not meant to be official or perfect. It’s simply a personal project that mixes nostalgia with modern technology, giving the original Xbox experience a new life on today’s hardware.
